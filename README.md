# V2RayN 全能采集器

一个强大的V2Ray节点采集工具，支持多种采集模式和协议，具有美观的用户界面和完善的功能。

## ✨ 主要功能

- **双模式采集**
  - 直连模式：直接从指定的URL采集节点链接
  - 论坛模式：从论坛页面扫描并提取订阅链接

- **智能节点提取**
  - 支持多种协议：vmess, vless, trojan, ss, hysteria2
  - 智能正则匹配，自动提取节点
  - Base64自动解码
  - HTML标签清理和过滤

- **完善的配置管理**
  - 可视化配置管理界面
  - 支持添加/删除直连源和论坛源
  - 可自定义正则表达式
  - 支持导入/导出配置

- **美观的用户界面**
  - 专业的菜单系统
  - 实时日志显示
  - 结果预览和导出
  - 支持窗口自由缩放

## 🚀 技术特点

- 多线程采集，不阻塞界面
- 浏览器伪装，避免被识别
- SSL证书验证可选
- 智能错误处理和日志记录
- 支持多种协议的节点链接

## 📦 安装方法

### 环境要求

- Python 3.7+
- 依赖库：
  - tkinter
  - requests
  - urllib3

### 安装步骤

1. 克隆或下载本项目
2. 安装依赖库：
```bash
pip install requests urllib3
```

3. 运行程序：
```bash
python v2rayN_manager_美化版.py
```

## 📖 使用说明

### 基本使用

1. 选择采集模式（直连或论坛）
2. 选择数据源
3. 点击"开始采集"按钮
4. 等待采集完成
5. 复制或导出结果

### 菜单功能

- **文件菜单**
  - 导出结果：将采集结果保存到文件
  - 导入配置：从JSON文件导入配置
  - 退出：退出程序

- **编辑菜单**
  - 清空结果：清空采集结果
  - 清空日志：清空执行日志
  - 复制结果：复制所有结果到剪贴板

- **工具菜单**
  - 配置管理：打开配置管理窗口
  - 刷新源列表：刷新数据源列表

- **帮助菜单**
  - 使用说明：详细的使用指南
  - 关于：软件信息和功能介绍
  - 检查更新：检查软件更新

### 配置管理

点击"配置管理"按钮可以：
- 添加/删除直连源
- 添加/删除论坛源
- 编辑正则表达式

### 快捷操作

- 右键点击日志可以复制、全选或清空
- 使用菜单栏的"导出结果"保存采集结果

## 🔧 配置说明

配置文件为 `v2ray_pro_config.json`，包含以下配置项：

- `direct_sources`: 直连源列表
- `forum_sources`: 论坛源列表
- `protocols`: 正则表达式匹配规则
